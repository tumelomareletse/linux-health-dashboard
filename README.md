# Linux System Health Dashboard

A Python tool that monitors Linux system health in real time.
Built for ITLSA1-T22 (Linux OS) at Eduvos, BSc IT: Software Engineering.

## What it does
- Shows CPU usage
- Shows RAM and swap usage
- Shows disk usage
- Lists top 5 processes by CPU
- Shows network interfaces

## How to run it
```bash
python3 dashboard.py
```
